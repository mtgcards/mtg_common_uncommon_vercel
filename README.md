# MTG 高価なコモン・アンコモンカード一覧

Scryfall API を使って MTG のコモン・アンコモン・基本土地・トークン/紋章カードの中から高値のものを一覧表示する静的 Web アプリ。

## 機能

- 発売年代別タブでカードを絞り込み
- エキスパンション別にセクション分けして表示（発売日昇順）
- カード画像・USD 価格を表示
- カード名クリックで晴れる屋の検索ページを新しいタブで開く
- セクションナビゲーション（エキスパンションシンボル付き）
- ページ上部へ戻るボタン

## 除外セット

以下のセットは結果から除外しています。

- Alpha / Beta / Unlimited などの旧版レギュラー
- Summer Magic / Edgar、Foreign Black Border など非流通品
- Planechase / Archenemy 系（ゲームコンポーネント扱い）
- `Duel Decks:` で始まるセット（全て）
- `Duel Decks Anthology:` で始まるセット（全て）
- `Archenemy:` で始まるセット（全て）
- Secret Lair / Promos / Mystery Booster 2 など

## 使い方

サーバー不要。`index.html` をブラウザで開くだけで動作します。

```
open index.html
```

## 使用技術・データソース

- [Scryfall API](https://scryfall.com/docs/api) — カードデータ・画像
- [晴れる屋](https://www.hareruyamtg.com/) — カード名クリック時の検索リンク先
- 純粋な HTML / CSS / JavaScript（フレームワーク・ビルドツール不使用）
